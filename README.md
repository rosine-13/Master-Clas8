# Application de Traduction avec API

## Description
Cette application web permet de traduire un texte du français vers plusieurs langues grâce à une API de traduction.

## Technologies utilisées
- HTML
- CSS
- JavaScript
- API MyMemory Translation

## Fonctionnalités
- saisie d'un texte
- choix de la langue
- traduction automatique
- interface simple

## API utilisée
MyMemory Translation API permet de traduire un texte automatiquement grâce à un service web accessible par requête HTTP.
